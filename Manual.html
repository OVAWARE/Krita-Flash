KritaFlash Manual

Introduction

KritaFlash is a powerful plugin for Krita that leverages Google's Gemini AI to transform your artwork with intelligent image generation. With just a text prompt, you can apply creative transformations to your existing artwork.

Example Transformation: Original artwork → Transformed with prompt "Convert to pencil sketch style"

Installation

  1. Download the KritaFlash plugin package.
  2. Enable the plugin by going to Settings → Python Plugin Manager → KritaFlash.
  3. Restart Krita.
  4. Ensure that the KritaFlash docker is enabled.

Setting Up Your API Key

KritaFlash requires a Google Gemini API key to function. This key is not provided with the plugin and must be obtained from Google AI Studio.

Getting a Gemini API Key

  1. Visit https://aistudio.google.com/app/apikey.
  2. Sign in with your Google account.
  3. Click on "Create API Key" button.
  4. Copy the generated API key to your clipboard.

Important: Keep your API key private. Do not share your key with others or include it in publicly accessible code.

Configuring the API Key in KritaFlash

  1. In Krita, open the KritaFlash docker.
  2. Click the "Configure API Key" button.
  3. Paste your Gemini API key into the dialog box.
  4. Click "Save".

Alternative Configuration Methods

You can also set up your API key as an environment variable named GEMINI_API_KEY. This method is preferable for system-wide configuration or if you're working in a shared environment.

Operating System Setting Environment Variable
Windows setx GEMINI_API_KEY "your_api_key_here" (Command Prompt)
Or via System Properties → Environment Variables
macOS/Linux Add export GEMINI_API_KEY="your_api_key_here" to your .bashrc or .zshrc file

Using KritaFlash

Basic Usage

  1. Open or create an image in Krita.
  2. Make sure the KritaFlash docker is visible (if not, go to Settings → Dockers → KritaFlash).
  3. Enter a descriptive prompt in the text box, such as "convert to oil painting style" or "add a cosmic background".
  4. Click the "Generate Image" button.
  5. Wait for the processing to complete.
  6. The generated image will be added as a new layer in your document.

Prompt Tips

Effective Prompts: Be specific and descriptive in your prompts. Instead of "make it better" try "enhance contrast and add subtle purple lighting from the top right".

Example Prompts Description
"Convert to watercolor style with soft edges" Transforms the image to simulate watercolor painting
"Add dramatic lighting from the left side" Enhances the image with directional lighting effects
"Make it look like a scene from cyberpunk future" Applies futuristic cyberpunk aesthetic
"Add a forest background with fog" Replaces or adds a new environment around your subject
"Color in the image" Attempts to color the image

Troubleshooting

Common Issues

Problem Possible Solution
"API key not configured" error Click the "Configure API Key" button and enter a valid Gemini API key.
Generation fails or times out
  • Check your internet connection
  • Verify your API key is valid
  • Try a simpler prompt
  • Try a smaller image size
Generated image doesn't appear as a new layer Check that your document is properly saved and you have sufficient permissions to create files in the output directory.
KritaFlash docker doesn't appear Go to Settings → Dockers and make sure KritaFlash is checked.

API Usage Limitations

Google Gemini API has usage limits based on your account tier. Free tier users may have restrictions on the number of requests per minute or per day. Check the Google AI pricing page for current limits and pricing information.
